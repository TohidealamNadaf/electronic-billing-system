# Nadaf Billing System

A desktop-based billing and estimate management system for Nadaf Furnitures, built with Angular (Frontend), NestJS (Backend), and Electron.

## Features
- Invoice Generation and Management
- Estimate Creation and Tracking
- PDF Print and Preview (Pop-out Window)
- Native Pinch-to-Zoom Support

## Tech Stack
- **Frontend**: Angular
- **Backend**: NestJS (SQLite)
- **Desktop Wrapper**: Electron
